# 🔐 Secure Login System

A secure web-based authentication system built using **Flask**, **SQLite**, and **Flask-Bcrypt**. This project demonstrates secure user authentication by implementing password hashing, session management, and protected routes.

---

## 📌 Project Objective

Develop a secure login system that:

- Allows users to register and log in securely.
- Stores user credentials in a SQLite database.
- Hashes passwords using **Flask-Bcrypt** before storing them.
- Prevents unauthorized access using session management.
- Protects user credentials from being stored in plain text.
- Uses SQLAlchemy ORM for secure database operations.
- Provides a simple and user-friendly interface.

---

## ✨ Features

- 👤 User Registration
